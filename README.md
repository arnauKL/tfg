# Treball Final de Grau (TFG): 

Final year BME thesis on *Deep Learning-Based Classification of Parkinson's
Disease Stages Using DaTSCAN*.


## Building

Typeset using [typst](https://typst.app/). Install the compiler and use the
provided Makefile:

```sh
git clone https://github.com/arnauKL/tfg
cd writing
make
```
